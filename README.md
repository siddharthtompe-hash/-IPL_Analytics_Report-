📊 IPL Data Analysis Dashboard (2008–2025)

This project presents an end-to-end IPL (Indian Premier League) data analysis using advanced data analytics and visualization techniques. The dashboard provides deep insights into team performance, player achievements, and season-wise trends from 2008 to 2025.

🚀 Project Highlights
📅 Covers 18 IPL seasons (2008–2025)
🗂️ Built using 4 relational datasets
📈 Processed over 278,206+ rows of data
📊 Interactive dashboard with dynamic filters (season-wise analysis)
⚡ Cleaned and transformed raw data using data modeling techniques
📌 Key Metrics & Insights
🏏 Total Matches Analyzed: 74 (2025 season snapshot)
